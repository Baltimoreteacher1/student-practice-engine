# Shared Archive

Use this folder for preserved local artifacts that should remain available but should not clutter the active production surface.

## What belongs here

- local one-off scripts
- scratch prompts
- preview renders
- preserved classroom artifacts that are not active workflow entrypoints

## What does not belong here

- active generators
- canonical workflow docs
- gold standards
- reusable templates

## Working rule

Anything moved here is preserved, not promoted.

Do not assume archive contents are parameterized, portable, or production-safe. If something from this folder becomes useful again, clean it up first and then move it into the appropriate canonical workflow area.
