# Shared Assets

Use this folder for reusable assets that support more than one workflow.

Recommended use:

- `templates/`: reusable PPTX, DOCX, Sheets, or Apps Script templates
- `gold-standards/`: shared reference artifacts that inform multiple systems
- `reference-inputs/`: scrubbed source inputs reused for testing
- `reference-outputs/`: approved outputs reused for comparison

Legacy assets at workspace root were preserved in place for safety and can be promoted here when they are clearly reusable.
