# Style Rules

## Writing expectations

- Output polished final teacher-facing writing.
- Keep language concise, concrete, and observation-ready.
- State what the teacher does, what students do, and what evidence to watch for.
- Preserve the lesson's original mathematical meaning, vocabulary, and rigor.
- Do not use draft framing, placeholders, emojis, or AI/meta commentary.

## Formatting expectations

- DOCX output is the default teacher-facing format.
- Use Calibri 11 with 1.15 line spacing.
- Use clean section headers.
- Use bold table headers where tables are appropriate.
- Use simple black table borders.
- Keep spacing readable and uncluttered.

## Source fidelity expectations

- Prefer direct source wording for the opening, guided work, independent tasks, closure, and discussion prompts.
- Preserve Be Curious / Notice-Wonder prompts, mindset prompts, worked examples, and Reveal Math workbook references when present.
- Preserve decomposition/composition, base/height, diagonal/perpendicular, missing-dimension, and formula-development language when present.
- Do not replace source problems with generic filler.
