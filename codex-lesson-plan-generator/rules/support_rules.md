# Support Rules

## Approved SPED supports

- `Profile A`: sentence starters, word banks, expressive language support
- `Profile B`: reduced distractions, manipulatives, extended time
- `Profile C`: breaks, chunking, graphic organizers, word banks
- `Profile D`: graphic organizers, text-to-speech, visuals, repetition
- `Profile E`: preteach vocabulary, highlight tool, attention strategies, small group
- `Profile F`: clarified/repeated directions, text-to-speech, visuals, immediate feedback
- `Profile G`: simplified language, word banks, graphic organizers, calculator, small group

## Support insertion rules

- Use only the approved support mapping above.
- Keep supports concise and classroom-usable.
- Embed support moves naturally across lesson phases instead of dropping all support language at the end.
- Do not change the source task in order to add supports.

## ESOL rules

- Add simple ESOL supports only when the source lesson includes vocabulary, discussion, or written explanation demands.
- Prefer student-friendly vocabulary definitions, visual supports, sentence starters, partner talk prompts, highlighted key words, and repeated directions.
- Do not add bilingual support unless the workflow explicitly supports it elsewhere.
